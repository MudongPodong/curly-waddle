
***어플 사용방법***
어플의 기능을 사용하기 위한 절차를 설명한다.
해당 설명에서 클릭은 터치와 동일한 의미로 사용된다.
어플을 실행시 먼저 AIO의 로고가 잠시 출력되고 로그인 화면으로 이동된다.
테스트에 자주 사용된 계정은
아이디 : aaa
비밀번호 : aaa
로써 검색기록, 구매이력, 최근 본 상품 등의 정보들이 가장 많은 계정이다.

1. 회원가입
   (1) 로그인 화면에서 'JOIN MEMBERSHIP'버튼을 클릭하여 회원가입 화면으로 이동한다.
   (2) 회원가입 화면에서 아이디, 비밀번호, 비밀번호 확인, 이름, 전화번호, 도로명 주소, 상세 주소를 입력한다.(AIO에서는 "0-답변"이라는 문자열이 포함된 아이디는 사용할 수 없다. 해당 문자열이 포함된 아이디를 입력 후 (3)을 진행하면 해당 문자열이 포함된 아이디는 사용할 수 없다는 메세지가 출력된다.)
   (3) '가입'버튼을 클릭해 회원가입을 진행한다. 이때 비밀번호와 비밀번호 확인이 다르다면 비밀번호가 일치하지 않다는 메세지가 뜬다.
	(또한 모든 항목에 처음과 끝에 공백(띄움)을 입력해도 없는것으로 취급하며 아무런 값을 입력하지 않거나 공백만을 여러번 입력해한 경우 값을 입력하라는 메시지가 출력 된다.)
   (4) 이미 존재하는 아이디라면 이미 존재하는 아이디라는 메세지가 출력된다. 아니라면 회원가입이 정상적으로 진행되고 회원가입 성공 메세지가 출력된다.
   (5) 회원가입 성공시 다시 로그인 화면으로 이동된다.
   (6) 회원가입 절차 도중 취소를 원한다면 '취소'버튼을 클릭하면 로그인 화면으로 이동된다.

2. 로그인
   (1) 본인이 회원가입시 입력한 아이디와 비밀번호를 입력하고 'LOGIN'버튼을 클릭한다.
   (2) 본인이 입력한 아이디가 DB에 존재하고 비밀번호가 DB와 일치한다면 로그인이 되어 홈 화면으로 이동한다.
   (3) 입력한 아이디가 DB에 존재하지 않는 경우 해당 아이디로된 계정이 없다는 메시지가 출력되고 비밀번호가 잘못된 경우 비밀번호를 확인해 달라는 메세지가 출력된다.

3. 자동 로그인
   (1) 로그인과 동일하게 아이디와 비밀번호를 입력한다.
   (2) 로그인 화면의 '자동 로그인' 체크박스를 체크한다.((1)과 (2)는 순서가 바뀌어도 상관없다.)
   (3) 'LOGIN'버튼을 클릭하여 로그인과 동일한 절차를 수행한다.(로그인 실패 시 자동 로그인 정보가 기록되지 않는다.)

4. 로그아웃
   (1) 바텀 네비게이션이 있는 화면(카테고리 화면, 검색 화면, 홈 화면, 즐겨찾기 화면, 마이페이지)에서 바텀 네비게이션의 마이페이지(최우측)로 이동한다.
   (2) '로그아웃'버튼을 클릭한다.
   (3) 로그아웃이 되며 자동 로그인 정보가 사라지며 로그인 화면으로 이동한다.(이때 로그인에 사용한 아이디, 비밀번호, 자동 로그인 체크 여부는 그대로 입력되어있다.)

5. 랜덤 상품 제안
   (1) 홈 화면의 '랜덤 상품 소개'라는 문구 아래로 자동으로 랜덤 상품이 5개 출력된다.(랜덤이므로 홈 화면으로 이동할때 마다 새로운 제품이 출력되며. 리스트로 보이기 때문에 위아래로 스크롤 하여 5개의 상품을 볼 수 있다.)
   (2) 해당 상품을 클릭하면 해당 상품의 상세페이지로 이동한다.

6. 제품 검색
   (1) 홈 화면 또는 바텀 네비게이션이 있는 화면(카테고리 화면, 검색 화면, 홈 화면, 즐겨찾기 화면, 마이페이지)에서 바텀 네비게이션의 검색 화면(왼쪽에서 2번째)으로 이동한다.
   (2) 홈 화면 상단 또는 검색 화면 하단의 검색바에 원하는 검색어를 입력한다.
   (3) 검색바 우측의 돋보기 버튼을 클릭하면 입력한 검색어로 검색이 되며 검색 결과 화면으로 이동한다.(공백 또는 공백만으로 이루어진 검색어로 검색시도 시 검색어를 입력해 달라는 메세지가 출력된다.)
   (4) 검색 결과 화면에 출력된 제품들은 리스트 형태로 위아래로 스크롤 하여 여러 상품을 볼 수 있으며 각 상품의 사진, 이름, 가격, 판매량을 확인할 수 있다.
   (5) 원하는 상품을 클릭하면 해당 제품의 상세 페이지로 이동한다.

7. 이벤트 소개
   (1) 홈 화면의 검색바 아래에 현재 AIO에서 진행 중인 이벤트의 이미지가 출력된다. 좌우로 스크롤하여 다른 이벤트도 확인 가능하다.

8. 제품 카테고리 검색
   (1) 바텀 네비게이션이 있는 화면(카테고리 화면, 검색 화면, 홈 화면, 즐겨찾기 화면, 마이페이지)에서 바텀 네비게이션의 카테고리 화면(최좌측)으로 이동한다.
   (2) 카테고리 화면에는 AIO의 총 12가지의 카테고리 버튼들이 존재하고 검색을 원하는 카테고리를 클릭한다.
   (3) 해당 카테고리에 해당하는 제품들이 검색되며 검색 결과 화면으로 이동한다.

9. 검색 제품 정렬
   (1) 검색 결과 화면에서 좌측 상단의 '정렬'버튼을 클릭한다.
   (2) 가격 순, 판매 순, 신상품 순으로 구성된 팝업이 출력되고 원하는 정렬을 클릭한다.(기본은 출력은 이름 순이며 이름 순은 정렬 팝업에 존재하지 않는다.)
   (3) 클릭한 정렬 순서대로 검색 결과 화면이 재출력된다.(가격은 싼 순서, 판매는 많이 팔린 순서, 신상품은 최신 등록 순서이다.)

10. 제품 상세 정보 확인
   (1) 제품 상세 페이지에서 제품의 상세 정보를 확인할 수 있다.
   (2) 위아래로 스크롤하여 큰 사진, 이름, 가격, 판매자, 상세 이미지, 상세 설명을 확인할 수 있다.
   (3) 상단의 큰 사진의 경우 좌우로 스크롤하여 여러 사진을 확인할 수 있다.

11. 즐겨찾기 등록
   (1) 제품 상세 페이지에서 좌측 하단의 속이 빈 하트버튼을 클릭하면 즐겨찾기에 등록된다.
   (2) 즐겨찾기 화면에서 속이 빈 하트버튼을 클릭하면 즐겨찾기에 등록된다.(이 경우 즐겨찾기에 등록한 제품을 즐겨찾기 화면에서 속이 찬 하트버튼을 눌려 해제한 경우 바로 다시 즐겨찾기 할 수 있는 것이다. 아래의 즐겨찾기 해제 참조)

12. 즐겨찾기 해제
   (1) 제품 상세 페이지에서 좌측 하단의 속이 찬 하트버튼을 클릭하면 즐겨찾기에서 해제된다.(이미 즐겨찾기를 등록한 경우 가능)
   (2) 즐겨찾기 화면에서 속이 찬 하트버튼을 클릭하면 즐겨찾기에서 해제된다.(이 경우 즐겨찾기가 해제되지만 즐겨찾기 화면에서 나가기 전까지는 즐겨찾기 화면 리스트에 해당 제품이 남아있으며 하트버튼이 속이 빈 하트버튼으로 변경된다. 이 버튼을 다시 누르면 다시 즐겨찾기에 등록된다.)

13. 즐겨찾기 확인
   (1) 바텀 네비게이션이 있는 화면(카테고리 화면, 검색 화면, 홈 화면, 즐겨찾기 화면, 마이페이지)에서 바텀 네비게이션의 즐겨찾기 화면(우측에서 2번째)으로 이동한다.
   (2) 현재 사용중인 아이디로 즐겨찾기 등록한 제품들이 리스트로 출력된다. 위아래로 스크롤 하여 해당 제품들을 확인할 수 있다.
   (3) 리스트에서 제품을 클릭하면 해당 제품의 상세 페이지로 이동한다.

14. 사용자 정보 수정
   (1) 바텀 네비게이션이 있는 화면(카테고리 화면, 검색 화면, 홈 화면, 즐겨찾기 화면, 마이페이지)에서 바텀 네비게이션의 마이페이지(최우측)로 이동한다.
   (2) '회원 정보 수정'버튼을 클릭하여 회원정보 수정 화면으로 이동한다.
   (3) 회원정보 수정 화면의 새 비밀번호와 비밀번호 확인 항목을 제외한 전화번호, 도로명 주소, 상세 주소는 DB의 사용자의 정보에서 가져와 자동으로 채워져 있다.
   (4) 수정을 원하는 항목을 수정한다. 이때 비밀번호를 바꾸고 싶으면 새 비밀번호와 비밀번호 확인을 입력한다.(이때 아무 값을 입력하지 않는 경우 수정되지 않는다. 예를 들어 새 비밀번호와 비밀번호 확인, 도로명 주소는 비워두고 전화번호와 상세 주소의 값만 수정한 경우 전화번호와 상세 주소만이 수정된다.)
   (5) 항목 입력 후 '회원 정보 수정'버튼을 클릭한다.(새 비밀번호와 비밀번호 확인에 입력한 것이 다른 경우 비밀번호가 일치하지 않다는 메세지를 출력한다.)
   (6) 수정이 완료되면 다시 마이페이지로 이동한다.(비밀번호를 변경한 경우 자동 로그인의 비밀번호 정보도 바뀌어 자동 로그인이 유지된다.)
   (7) 도중에 취소를 원하면 '취소'버튼을 클릭한다.

15. 사용자 정보 조회
   (1) 마이페이지의 '회원 정보 조회'버튼을 클릭하여 회원정보 조회 화면으로 이동한다.
   (2) 회원정보 조회 화면에서 해당 사용자의 고객명, 아이디, 전화번호, 주소가 출력된다.

16. 회원탈퇴
   (1) 마이페이지의 '회원탈퇴'버튼을 클릭한다.
   (2) 탈퇴를 확인하는 팝업이 출력되고 '확인'버튼을 클릭하면 DB에서 해당 계정이 삭제된다.(이때 계정은 삭제되고 작성했던 리뷰, 고객질의는 남게된다.)
   (3) 취소를 원하면 '아니오'버튼을 클릭한다.

17. 최근 본 상품 조회
   (1) 마이페이지의 '최근 본 상품'버튼을 클릭하여 최근 본 상품 화면으로 이동한다.
   (2) 최근 본 상품 화면에서 최근에 제품 상세페이지로 이동했던 제품들의 리스트가 출력된다.
   (3) 최근 본 상품은 최대 20개가 출력되며 위아래로 스크롤하여 확인 가능하다.
   (4) 리스트에서 원하는 제품을 클릭시 해당 제품의 상세페이지로 이동한다.

18. 환불 요청
   (1) 마이페이지의 '구매 이력 조회'버튼을 클릭하여 구매 이력 화면으로 이동한다.
   (2) 구매 이력 화면의 구매한 제품 리스트에서 환불을 원하는 제품의 '환불 신청'버튼을 클릭하여 환불 신청 화면으로 이동한다.
   (3) 환불 신청 화면에서 사유, 전화번호를 입력한다.
   (4) '환불 신청'버튼을 클릭하여 환불 신청을 한다.(이때 사유와 전화번호가 입력되지 않았다면 빈칸을 입력해 달라는 메세지가 출력된다. 아니라면 환불신청이 되었다는 메세지가 출력된다. 또한 이미 환불 신청을 하여 신청 이력(제품, 사용자 정보)이 DB남아있는 경우 이미 신청한 항목이라는 메세지가 출력된다.)
   (5) 환불 신청이 되면 DB에 환불 정보가 저장된다.(AIO의 직원이 환불 사유를 확인하고 전화번호를 통해 연락하여 환불 가능 여부를 알려주고 환불이 완료되는 경우 구매 이력에서 해당 제품을 지운다.(실제 AIO가 운영되지않기 때문에 DB에 기록하고 환불 절차는 진행하지 않는다.))

19. 구매 이력 조회
   (1) 마이페이지의 '구매 이력 조회'버튼을 클릭하여 구매 이력 화면으로 이동한다.
   (2) 구매 이력 화면의 구매한 제품 리스트를 통해 구매한 제품들의 사진, 구매한 날짜, 이름, 가격을 확인할 수 있다.(최근 구매한 상품일수록 리스트의 윗부분에 배치된다.)
   (3) 위아래로 스크롤하여 현재까지 구매한 모든 제품을 볼 수 있다.

20. 포인트 충전
   (1) 마이페이지의 'POINT CHARGE'버튼을 클릭하여 포인트 충전 화면으로 이동한다.
   (2) 포인트 충전 화면에서 충전에 사용할 은행, 계좌, 충전할 금액을 입력한다.(실제 AIO가 운영되지않기 때문에 실제 계좌를 통해 충전하지 않으므로 은행과 계좌는 아무런 값을 넣어도 동작하도록 해두었다.)
   (3) '충전'버튼을 클릭하여 충전을 진행한다.(금액은 공백인 경우 금액을 입력해 달라는 메시지가 출력된다.)
   (4) 충전이 완료되면 마이페이지로 이동하며 마이페이지의 현재포인트가 충전한 금액만큼 증가되어 출력된다.
   (5) 도중 취소를 원하면 '취소'버튼을 클릭하여 마이페이지로 돌아간다.

21. 현재 포인트 조회
   (1) 마이페이지의 상단 사용자 아이디 아래 회색부분에 현재의 포인트가 출력되며 이를 통해 현재 포인트를 조회할 수 있다.

22. 장바구니 등록
   (1) 제품 상세페이지에서 '장바구니담기'버튼을 클릭하여 장바구니에 등록한다.(등록시 장바구니에 담았다는 메시지가 출력된다. 이미 담긴 제품이라면 이미 장바구니에 담긴 제품이라는 메시지가 출력된다.)
   (2) 장바구니 화면에서 '재등록'버튼을 클릭하여 장바구니에 등록할 수 있다.(이 경우 장바구니에 이미 담겨있던 제품을 등록 해제하였을때 재등록 하기위한 것이다. 즐겨찾기 화면에서의 즐겨찾기 등록과 비슷한 경우)

23. 장바구니 조회
   (1) 마이페이지의 '장바구니'버튼을 클릭하여 장바구니 화면으로 이동한다.
   (2) 장바구니 화면에서 현재 장바구니에 담겨있는 제품들을 리스트로 확인할 수 있다.
   (3) 위아래 스크롤을 통해 리스트를 확인할 수 있다.

24. 장바구니 제품 구매
   (1) 장바구니 화면에서 '결제'버튼을 클릭하여 결제 화면으로 이동한다.
   (2) 결제 화면의 도로명 주소, 상세 주소, 전화번호, 수취인은 DB의 사용자 정보를 이용해 자동으로 채워져있다.
   (3) 수정을 원하는 항목의 경우 수정을 하고 '결제'버튼을 클릭하여 구매를 한다.(비어있는 항목이 있다면 해당 항목을 입력해 달라는 메세지가 출력된다.)
   (4) 이때 장바구니의 제품중 재고가 없는 제품이 존재한다면 장바구니에 품절제품이 존재한다는 메세지가 출력된다.
   (5) 장바구니가 비어있는 경우 결제할 제품이 없다는 메세지가 출력된다.
   (6) 장바구니의 제품들의 가격의 합이 포인트보다 높은경우 포인트가 부족하다는 메세지가 출력된다.
   (7) 구매가 완료되면 구매가 완료되었다는 메세지가 출력되며 마이페이지로 이동된다.

25. 장바구니 삭제
   (1) 장바구니 화면에서 장바구니에서 삭제를 원하는 제품의 '삭제'버튼을 클릭한다.
   (2) 해당 제품이 장바구니에서 등록이 해제되고 '삭제'버튼은 '재등록'버튼으로 변경된다.(DB에서는 삭제되지만 한번 장바구니 화면에서 벗어나기 전까지는 장바구니 리스트에 출력된다. 재등록 버튼을 클릭하면 다시 장바구니에 등록된다.)

26. 리뷰 조회
   (1) 제품 상세페이지에서 'REVIEW'버튼을 클릭하여 리뷰 화면으로 이동한다.
   (2) 자신이 리뷰를 등록한 제품이라면 자신의 리뷰가 상단에 출력되고 아니라면 해당 공간이 비워져 있다.
   (3) 하단에서는 다른 사용자들이 작성한 리뷰들을 조회할 수 있다.(위아래 스크롤을 통해 확인가능)
   (4) 리뷰는 리뷰 내용과 해당 제품에대해 긍정적인지 부정적인지를 나타내는 그림이 있다.

27. 리뷰 추가
   (1) 리뷰 화면에서 자신의 리뷰가 등록되어있지 않다면 'UPLOAD REVIEW'버튼이 존재하고 해당 버튼을 클릭하여 리뷰 추가 화면으로 이동한다.(만약 제품을 구매하지 않았다면 제품을 먼저 구매하라는 메시지가 출력되고 리뷰추가화면으로 이동되지않는다.)
   (2) 리뷰 추가 화면에서 리뷰 내용을 작성하고 해당 제품에 긍정적인지 부정적인지를 체크한 후 '등록'버튼을 클릭한다.(이때 리뷰란이 공백이면 리뷰를 입력해달라는 메세지가 출력된다. 정상적으로 등록되면 리뷰가 등록되었다는 메세지가 출력된다.)
   (3) 리뷰가 등록되고 리뷰 화면으로 돌아간다.(자신이 작성한 리뷰가 리뷰 화면에 추가된다.)
   (4) 취소를 원하면 '취소'버튼을 클릭한다.

28. 리뷰 수정 및 삭제
   (1) 리뷰 화면에서 자신의 리뷰가 등록되어 있다면 'MODIFY REVIEW'버튼이 존재하고 해당 버튼을 클릭하여 리뷰 수정 및 삭제 화면으로 이동한다.
   (2) 리뷰 수정 및 삭제 화면의 항목들은 DB의 리뷰 정보를 통해 자신이 입력한 리뷰의 내용으로 자동으로 채워져 있다.
   (3) 리뷰를 수정하고 '수정'버튼을 클릭하면 리뷰가 수정된다.(리뷰란이 공백이면 리뷰를 입력해 달라는 메세지가 출력된다. 수정이 되면 수정을 완료했다는 메세지가 출력된다.)
   (4) 리뷰의 삭제를 원하면 '삭제'버튼을 클릭한다.(삭제되면 삭제되었다는 메세지가 출력된다.)
   (5) 도중 취소를 원하면 취소 버튼을 클릭한다.
   (6) 수정 또는 삭제 후 리뷰 화면에서 해당 동작들에 따라 수정된 리뷰 화면이 출력된다.

29. 고객질의 등록
   (1) 마이페이지의 '고객상담'버튼을 클릭하여 고객질의 화면으로 이동한다.
   (2) 고객질의는 채팅과 같은 형태로 이루어지며 다른 사용자들의 질의도 확인할 수 있다. 하단의 바에 질의문을 작성한 후 우측의 삼각형의 버튼을 클릭하면 질의문이 DB에 저장된다.(공백을 입력한 경우 저장되지 않는다.)

30. 고객질의 조회
   (1) 마이페이지의 '고객상담'버튼을 클릭하여 고객질의 화면으로 이동한다.
   (2) 다른 사용자의 질의는 좌측에 초록색으로 표시되며 본인의 질의는 우측에 파란색으로 표시된다. AIO에서 단 답변은 주황색으로 우측에 표시된다.

31. 고객질의 삭제
   (1) 고객질의 화면에서 본인의 질의를 롱클릭하면 삭제 여부를 물어보는 팝업이 출력된다.(본인의 질의만 가능. 다른 사용자의 질의나 답변의 경우 불가능)
   (2) '확인'을 클릭하면 해당 질의가 삭제된다.
   (3) '아니오'를 클릭하면 취소된다.

32. 고객센터 정보 조회
   (1) 마이페이지의 '고객센터'버튼을 클릭하여 고객 센터 화면으로 이동한다.
   (2) 고객 센터 정보 화면에서 AIO고객센터의 이메일, 전화번호, 주소를 확인 가능하며 주소위치가 지도로 표시된다.

33. 최근 검색기록 조회
   (1) 검색 화면의 상단에서 최근 검색기록을 조회할 수 있다.
   (2) 최근에 검색된 검색어일수록 상단에 위치하며 최대 20개까지 표시된다.
   (3) 위아래 스크롤을 통해 확인 가능하다.
